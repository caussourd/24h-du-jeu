# <PROJECT_NAME>


## Installation

	$ git clone git@bitbucket.org:martinogden/<PROJECT_NAME>.git
	$ cd <PROJECT_NAME>
	$ ./bin/install


## Creating an admin superuser

	$ ./bin/createsuperuser


## Running development server

From the root of the project, run:

	$ ./bin/runserver

and go to http://127.0.0.1:8000/


## Tests

From the root of the project, run:

	$ ./bin/test
